📂 Sproyets - Gestor de Proyectos Universitarios

¡Bienvenido a Sproyets! Una plataforma web interactiva diseñada especialmente para que los estudiantes organicen sus trabajos académicos sin caos. Con un diseño moderno, minimalista y de alto contraste basado en la paleta de colores Blanco, Negro y Rojo, esta aplicación permite gestionar proyectos y tareas de forma dinámica utilizando una pizarra Kanban.

✨ Características Principales

Autenticación de Estudiantes: Sistema interactivo de inicio de sesión y registro con validación en tiempo real (simulado localmente).

Pestaña "Mis Proyectos": Un espacio privado donde se listan únicamente los proyectos creados y gestionados personalmente por el usuario logueado.

Pestaña "Lista de Proyectos": Una vista comunitaria donde se listan todos los proyectos del sistema, permitiendo la exploración de ideas y la simulación de colaboraciones.

Buscador Inteligente: Filtro en tiempo real en la lista global de proyectos para encontrar entregas o materias específicas instantáneamente.

Pizarra Kanban Interactiva:

Visualización de tareas divididas en tres estados: Pendientes, En Progreso y Completadas.

Interacción Drag & Drop (arrastrar y soltar) nativa para cambiar de estado las tareas.

Botones rápidos inferiores de apoyo en cada tarjeta para facilitar la navegación en dispositivos móviles o pantallas táctiles.

Detección visual automática de tareas retrasadas (resaltadas en color rojo si la fecha límite ha expirado).

Componentes de Interfaz Propios (UI/UX Premium):

Se eliminaron las molestas alertas nativas del navegador (alert y confirm).

Se añadieron ventanas flotantes de notificación (Toasts) elegantes y un modal de confirmación personalizado para eliminar tareas de forma segura.
